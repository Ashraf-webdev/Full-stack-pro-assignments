Full Stack Web Development Tech Stack

  1. HTML

    Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compitable with all web browser
    • Allow for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benefits
    CSS is used to control the presentation and styling of HTML elements.
    CSS provides tools for creating responsive and flexible page layouts.
    • CSS allows for the reuse of styles across multiple pages.
    • CSS enable responsive design
    • CSS offers built-in support for animations and transitions

  3. Javascript

    Primary Use Cases Key Features/Benefits
    adding interactivity and dynamic behavior to web pages
    • JavaScript can respond to user actions and trigger specific actions in response.
    • JavaScript can manipulate and modify the Document Object Model (DOM).
    • JavaScript is suitable for both small-scale and large-scale applications.
